# LaundroMagic - Site Vision

**Stitch Project ID:** 10041881444321079848

## 1. Overview

LaundroMagic is a comprehensive laundry shop management system with two distinct user portals:
- **Owner Portal** (Desktop): Full management dashboard for shop owners
- **Staff Portal** (Mobile): Quick order processing for employees

## 2. Target Users

| Role | Device | Primary Tasks |
|------|--------|---------------|
| Owner | Desktop | Dashboard, reports, inventory, staff management |
| Staff | Mobile | Create orders, update status, view order list |

## 3. Brand Values
